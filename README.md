# 🚀 AI-ассистент для командной работы (AI PM Assistant)

Прототип интеллектуального бота-помощника, который берет на себя роль project-менеджера. Работает в фоне, читает чаты, помогает фиксировать итоги встреч, ведет Kanban-доску, напоминает о дедлайнах и внедряет элементы геймификации.

## 🎯 Решаемая проблема
В рабочих чатах теряются задачи, дедлайны срываются, а ручное внесение задач во внешние системы убивает продуктивность. Наш бот обеспечивает **минимальное ручное управление**: он сам извлекает контекст, а человек только подтверждает результат.

## ✨ Ключевые возможности (MVP)

### 1️⃣ Умная интеграция с Telegram
- Распознает задачи, дедлайны (включая "до пятницы") и ответственных по `@username`.
- **Устойчивость к неполным данным:** если исполнитель не указан, задача автоматически попадает в 📥 **Бэклог** для ручного назначения администратором.
- Игнорирует обычный диалог, реагируя только на ключевые слова ("сделать", "нужно", "дедлайн" и т.д.).

### 2️⃣ Обработка итогов встреч (Meeting Workflow)
- Администратор отправляет боту текстовое саммари встречи.
- Бот предлагает **Inline-кнопки**: "✅ Подтвердить и в Канбан" или "❌ В корзину". Это реализует паттерн "Центрального окна" и исключает создание мусорных задач.

### 3️⃣ Встроенная Kanban-доска (Web UI)
- 4 колонки: Бэклог, Сделать, В работе, Готово.
- Drag-and-drop перемещение задач с мгновенным обновлением статуса.
- Редактирование задач (название, исполнитель, дедлайн, описание) через модальное окно.
- Генерация персонализированных советов от ИИ по кнопке (с поддержкой Markdown и прокруткой).

### 4️⃣ Проактивность и контроль
- **Напоминания о дедлайнах:** бот пишет в чат за 2 часа до срока, текая ответственного.
- **Вечерняя синхронизация:** по команде `/evening` бот агрегирует все незакрытые задачи и присылает дайджест по участникам, сверяя чат с Kanban-доской (без спама).

### 5️⃣ Геймификация (RPG-элементы)
- Начисление XP (+10) за создание задач.
- Публичный рейтинг команды с медалями 🥇🥈🥉 и уровнями прямо на Kanban-доске.

## 🛠 Технологический стек
- **Backend:** Python 3.10, FastAPI (асинхронный), SQLAlchemy 2.0
- **Bot:** Aiogram 3.x
- **Database:** PostgreSQL 15 (AsyncPG)
- **Frontend:** HTML5, Bootstrap 5, SortableJS (Drag-and-Drop), Marked.js (Markdown рендеринг)
- **AI:** OpenAI-compatible API (VSEGPT / OpenRouter / OpenAI)
- **Deployment:** Docker, Docker Compose

## 🏗 Архитектура решения
Проект построен по принципу легковесного оркестратора. Учитывая ограничения сервера (2 GB RAM), тяжелые вычисления (LLM-запросы) вынесены на внешние API. Бот отвечает за парсинг, бизнес-логику, работу с БД и UX, что гарантирует стабильность и масштабируемость.
